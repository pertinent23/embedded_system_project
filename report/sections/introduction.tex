\section{Introduction}
This report presents the design and implementation of a Smart Plant Watering System, an automated embedded solution developed to monitor and maintain soil moisture levels for domestic plants. The primary objective of this project is to provide a reliable, efficient, and user-friendly system that reduces the need for manual intervention while ensuring optimal plant health.

The system is built around the Microchip PIC16F1789 microcontroller, which serves as the central processing unit. By integrating a capacitive soil moisture sensor, a submersible pump, and a liquid crystal display (LCD), the system creates a closed-loop control environment. The sensor continuously monitors the soil condition, and the microcontroller triggers the irrigation process via a relay whenever the moisture level falls below a predefined threshold.

Throughout the development process, particular attention was paid to both hardware robustness and software efficiency. This project not only focuses on functional requirements but also explores the integration of human-machine interface (HMI) elements and power-saving strategies. The following sections detail the technical choices, circuit design, software architecture, and the final validation of the system.
