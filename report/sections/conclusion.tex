\section{Conclusion}
The Smart Plant Watering System project successfully demonstrates the integration of multiple embedded technologies into a cohesive and functional product. By utilizing the PIC16F1789's peripherals and implementing efficient control logic, we have created a system that is both reliable and user-friendly.

Key achievements of this project include:
\begin{itemize}
    \item Implementation of a robust hysteresis-based irrigation logic.
    \item Successful deployment of an I2C communication protocol via bit-banging.
    \item Development of an informative HMI with multiple display modes.
    \item Integration of the electronics into an aesthetic diorama.
\end{itemize}

Future improvements could include the integration of a Wi-Fi module (such as the ESP8266) to provide remote monitoring capabilities via a web interface, or the addition of a light sensor to prevent irrigation during the hottest parts of the day to reduce water evaporation.

In conclusion, this project has provided valuable experience in hardware-software co-design, real-time interrupt management, and signal processing in a practical embedded context.
